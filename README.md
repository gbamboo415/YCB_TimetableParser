# YCB_TimetableParser
横浜市交通局（横浜市営バス）時刻表ページの簡易パーサー

Rubyスクリプトとシェルスクリプトは同一出力

## 入力
横浜市交通局（横浜市営バス）経路・時刻表検索システム「hamaBus」の、各バス便停車時刻表HTMLファイル
